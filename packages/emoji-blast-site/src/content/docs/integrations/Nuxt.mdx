---
title: Nuxt
---

**[`@konami-emoji-blast/nuxt`](https://www.npmjs.com/package/@konami-emoji-blast/nuxt)** provides an [Nuxt module](https://nuxt.com/modules) that loads `konami-emoji-blast` on all pages of your site.

First install the `@konami-emoji-blast/nuxt` package as a dependency:

```shell
npm i @konami-emoji-blast/nuxt
```

You can then use the `"@konami-emoji-blast/nuxt"` plugin in your `nuxt.config.*` config file's `modules`:

```ts
export default defineNuxtConfig({
	modules: ["@konami-emoji-blast/nuxt"],

	// Temporary workaround for https://github.com/JoshuaKGoldberg/emoji-blast/issues/822:
	vite: {
		optimizeDeps: {
			include: ["konami-emoji-blast > konami-code-js"],
		},
	},
});
```

Then, pressing the [Konami Code](https://en.wikipedia.org/wiki/Konami_Code) on your Nuxt site should trigger fireworks blasts of emoji on the page.
