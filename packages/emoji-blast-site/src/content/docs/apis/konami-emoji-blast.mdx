[`konami-emoji-blast`](https://www.npmjs.com/package/konami-emoji-blast) is a package that combines two others:

1. [`konami-code-js`](https://www.npmjs.com/package/konami-code-js): detects when the user has entered the [Konami Code](https://en.wikipedia.org/wiki/Konami_Code)...
1. [`emoji-blast`](../emoji-blast): ...to start `emojiBlast()`s until the code is entered again.

If a user presses <em>up</em> <em>up</em> <em>down</em> <em>down</em> <em>left</em> <em>right</em> <em>left</em> <em>right</em> <em>b</em> <em>a</em>
on their keyboard or taps it on their mobile phone, emoji blasts will start firing on the page.
Triggering the code again stops the blasts.

### Quick Bundler Start

If you're writing JavaScript or TypeScript with a bundler, first install the `konami-emoji-blast` package as a dependency:

```shell
npm i konami-emoji-blast
```

You can then import it in your code to access its `initializeKonamiEmojiBlast` function:

```tsx
import { initializeKonamiEmojiBlast } from "konami-emoji-blast";

initializeKonamiEmojiBlast();
```

## API

`konami-emoji-blast` export a single function:

- [`initializeKonamiEmojiBlast`](#initializekonamiemojiblast): Initializes the Konami Code listener on the page.

### `initializeKonamiEmojiBlast`

```ts
import { initializeKonamiEmojiBlast } from "konami-emoji-blast";

initializeKonamiEmojiBlast();
```

This will create a new [`konami-code-js`](https://www.npmjs.com/package/konami-code-js) `KonamiCode` instance to listen for user input.
It will call [`emojiBlasts`](./emoji-blast#emojiblasts) whenever the code is detected.

`initializeKonamiEmojiBlast()` returns a cancellation function that cancels any pending work.

```typescript
// Commence code listening!...
const cancel = emojiBlasts();

// ...but not after ten seconds.
setTimeout(stop, 10000);
```

#### `initializeKonamiEmojiBlast` Options

`initializeKonamiEmojiBlast` may take a single optional parameter, `onActivate`.

##### `onActivate`

A function to run whenever the user inputs the Konami Code.

```ts
import { initializeKonamiEmojiBlast } from "konami-emoji-blast";

initializeKonamiEmojiBlast(() => {
	console.log("Commence blasting! ðŸŽ†");
});
```
